Neat Download Manager 免费多线程下载器

· 2022-01-16 · # 下载器

Neat Download Manager是一个简单而轻量级的GUI，围绕着一个强大而优化的下载引擎。它被设计为简单，易于使用，并且轻度占用硬件资源（CPU，Ram，Power等）。
Windows和Mac版本共享相同的下载引擎，该引擎以C++编写，可处理下载的所有详细信息：协议实现，连接和带宽控制，动态分割，合并段和...
浏览器扩展可以添加到Firefox，Chrome，MS Edge和其他基于Chromium的浏览器（Opera，Brave，Vivaldi等）中。该扩展将NeatDownloadManager与浏览器集成在一起，可以捕获下载链接并检测网站上的所有视频/音频链接。
使用优化的动态分割算法。
将下载速度提高到总可用带宽。
根据下载状态（完整、不完整）和文件类型（视频、文档等）组织下载
支持 HTTP、HTTPS 和 FTP 协议。
支持HTTP代理和SOCKS协议。
支持 HTTP 身份验证和代理身份验证（基本、摘要、NTLM）
具有暂停/恢复功能，可以恢复崩溃的下载。
具有浏览器扩展，可以向其发送下载链接，并帮助您从任何网站下载视频/音频。
允许您设置下载的带宽限制，即使下载正在进行中。
允许您设置下载的最大连接数，即使下载正在进行中。
可以下载HLS视频的所有.ts文件，并在最后将所有片段合并为一个.ts文件。
可以续订过期的下载。
适用于拖放（仅适用于Mac，观看视频剪辑）
