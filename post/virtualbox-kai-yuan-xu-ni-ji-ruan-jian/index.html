VirtualBox 开源虚拟机工具

· 2022-01-15 · # 虚拟机

VirtualBox 是一款功能强大的 x86 和 AMD64/Intel64虚拟化产品，适用于企业和家庭使用VirtualBox不仅是面向企业客户的功能极其丰富、高性能的产品，也是唯一一款根据GNU通用公共许可证（GPL）版本2条款作为开源软件免费提供的专业解决方案。
目前，VirtualBox在Windows，Linux，Macintosh和Solaris主机上运行，并支持大量客户机操作系统，包括但不限于Windows（NT 4.0，2000，XP，Server 2003，Vista，Windows 7，Windows 8，Windows 10），DOS / Windows 3.x，Linux（2.4，2.6，3.x和4.x），Solaris和OpenSolaris，OS / 2和OpenBSD。
